單元二 設定玩家  
 
人氣：
 
 

一個地圖上總不能只有小弟一個人在玩吧？ (像模擬城市？) 而且小弟如果想海扁電腦，也要有一堆資源當後盾，然後小弟就可以放一堆電腦的部隊殺個過癮。

在編輯器上方的「遊戲者」就是讓我們做上述設定的地方。這裡您可以決定地圖上要有幾個遊戲者，也可以設定每個遊戲者種族、種族名稱、顏色、一開始的時代及資源等。

這裡比較有趣的是「性格」這個項目，它可以設定電腦的人工智慧。例如當食物存量大於 100 時，才能生產村民等。在標準模式下，電腦就會依世紀帝國二內建的 AI 來行動；而選擇「無」時，電腦則是乖乖地作它該做的事 (例如採集食物、巡邏) 而不會有更進一步的行動 (如更改外交、升級等) 。還有，如果有使用到「凍結單位」這個觸發事件效果時，也記得要將性格設為「無」，至於觸發事件稍後小弟會再說明。

以下小弟來解說如何設定新的電腦性格：

  1. 建立兩個檔案： MyAI.ai 及 MyAI.per ，主檔名不一定要是 MyAI ，只要兩者主檔名相同即可。並將它們存放在安裝世紀帝國二目錄中的「AI」這個子目錄下。

  2. 用文字編輯器 (如記事本) 開啟 MyAI.per ，然後參考世紀帝國二光碟片中「DOCS」目錄下的 CPSB.doc ，編寫 MyAI.per 中的內容 (也可以參考這裡或以下的範例) 。至於 MyAI.ai 則保持空白。

  3. 存檔後，回到地圖編輯器，您會發現在遊戲者的性格那一欄多了一個「MyAI」的選項。

  4. 選擇「MyAI」，然後存檔測試。

要注意的是，只有電腦會受人工智慧設定檔的影響。

這裡小弟提供兩個簡單的範例供大家參考：

(defrule

	(food-amount greater-than 100)

	=>

	(train villager)	;分號後是註解

)

這是告訴電腦，當其存量大於 100 時，就開始生產村民。

(defrule

	(cheats-enabled)

	=> 

	(chat-to-all "作弊功能已開啟" )

	(disable-self)

)

這是可以偵測是否可以作弊。

世紀帝國二的人工智慧是一門相當龐大的課題，小弟本身也再鑽研中，希望有機會能撰寫這樣的內容供大家參考。
